Dart is a new language for scalable, structured web programming. Take a look and tell us what you think.

From quick prototypes to serious apps
Dart's optional types let you prototype quickly and then revise your code to be more maintainable.
Wherever you need structured code
You can use the same Dart code both in modern web browsers* and on servers.
Familiar yet new
Dart code should look familiar if you know a language or two, and you can use time-tested features such as classes and closures. Dart's new features—such as optional typing—make it easier for you to develop and maintain software.
Tools included
The Dart SDK includes the Dart Editor, a VM, core libraries, and more.
