A new platform for structured web apps

You can use the Dart language and libraries for all web apps, large and small. Dart is still in preview, so try it out and tell us what you think.

Dart apps work on servers, on browsers with a built-in Dart VM, and (thanks to Dart-to-JavaScript compilation) on modern web browsers. Here's the current state of Dart support:
