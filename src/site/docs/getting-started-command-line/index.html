Getting Started: Hello, Command Line

Follow this tutorial on a Linux or Mac OS system to write and run a Dart program using command-line tools.

Only have Windows? Sorry, there's no command-line interface yet. You can use the IDE instead, as described in Getting Started: Hello, World.

Step 1: Download and install the Dart bundle

  1. Download the Dart bundle: Download
  2. Unzip the bundle by double-clicking it in a file viewer or by executing a command in a shell window: 
    mv dart.zip ~/bin
unzip dart.zip
  3. Note the location of the dart_bin {PENDING: check tool name} executable file. For convenience, create a shortcut to it or add its directory to your path.

The Dart bundle contains tools such as:

  • A compiler that translates Dart to JavaScript
  • An Eclipse-based integrated development environment (IDE)
  • A virtual machine (VM), which you can use to run Dart code without first translating it to JavaScript
Note: This tutorial uses only the virtual machine from the bundle, to keep things simple. Unfortunately, the Dart VM only works on Linux and Mac OS, for now. {PENDING: point to more info}

Dart is object oriented.
