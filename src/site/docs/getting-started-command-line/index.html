Getting Started: Hello, Command Line

Follow this tutorial on a Linux or Mac OS system to write and run a Dart program using command-line tools.

Only have Windows? Sorry, there's no command-line interface yet. You can use the Dart Editor instead, as described in Getting Started: Hello, World.

Step 1: Download and install the Dart bundle

  1. Download the Dart bundle: Download
  2. Unzip the bundle by double-clicking it in a file viewer or by executing a command in a shell window: 
    mv dart.zip ~/bin
unzip dart.zip
