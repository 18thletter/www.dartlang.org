This program illustrates a few things about Spot that are different from JavaScript:

Spot is object oriented.
As a rule, Spot source code has methods in classes, instead of functions. In hello.spot, the HelloWorld class has one method, named main().

Spot lets you use types—but you don't have to.
In hello.spot, the main() method has the type void, indicating that the method doesn't return a value. However, the arguments parameter has no type (although it could).
Spot source filenames end in .spot.
