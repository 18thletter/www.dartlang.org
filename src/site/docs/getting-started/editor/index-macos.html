Follow this tutorial to install and use Dart Editor on Mac OS X.

Dart Editor is a lightweight, open-source editor based on Eclipse components. In addition to editing Dart programs, you can use Dart Editor to invoke the Dart-to-JavaScript compiler and launch Dart-based web apps.

Note: These instructions are also available for Windows and {% comment %} Mac. {% endcomment %} Linux.

Step 1: Download and install the software

In this step, you'll install Dart Editor and, if necessary, a Java runtime environment.

  1. Download the Dart Editor ZIP file:
    Dart Editor ZIP file
    Want the latest version? To try out the most recent, untested version of Dart Editor, download this instead: Dart Editor ZIP file (latest)

  2. Unarchive the ZIP file by double-clicking it.

  3. Put the resulting directory somewhere convenient, such as /Users/me/dart.

    We'll call this your Dart installation directory. This installation directory contains the DartEditor executable file and several subdirectories, including a samples directory.

    Important: Do not use the Dart installation directory for any other Eclipse-based software. If Dart Editor and Eclipse share the same workspace directory, they'll save conflicting settings, resulting in warnings and errors from both Dart Editor and Eclipse.

  4. If you don't already have a Java runtime, download and install it.

    • If you're using OS X Lion, see the download instructions at Java for OS X Lion.
    • If not, you're fine; Java is pre-installed on OS versions before Lion.

Step 2: Launch the editor

Go to your Dart installation directory, and double-click DartEditor.

Is startup slow?

If you use Sophos Anti-Virus, Dart Editor (like most Java applications) starts up 10-15 times slower than normal. The fix is to turn off scanning of the app's JAR files. For example:

  1. In Sophos Anti-Virus preferences, choose On-access Scanning.
  2. If the preferences are locked, click the lock and enter your password.
  3. Click Excluded Items and then the + sign.
  4. Navigate to the directory that contains DartEditor.
  5. Search for files under that directory with names that end in .jar.
  6. Select all these .jar files, and click Open.
