Step 1: Download and install the software

In this step, you'll install the Dart Editor and version 7 of the JDK.

{PENDING: If the JDK is still a sticking point on 10/10, flesh out these instructions and perhaps tell them how to test whether they already have the right version of the JDK.}

  1. Download the Dart Editor that matches your system architecture:
    • 32-bit Dart Editor
    • 64-bit Dart Editor

    {PENDING: those links don't work; for now, use http://www.corp.google.com/~messick/no_crawl/dart/

  2. Unarchive the file. For example: 
     $ tar xvf DartBuild-linux.gtk.x86_64.zip
  3. Download JDK 7 for your system architecture from this page:
    Java SE Development Kit 7 Downloads

  4. Install the JDK somewhere, such as /home/me/jdk1.7.0.

  5. Put this JDK in your path, so that it'll be the first Java environment found when you execute DartEditor. For example: 
    PATH=/home/me/jdk1.7.0/bin:$PATH
