Follow this tutorial to install and use Dart Editor on Linux.

Dart Editor is a lightweight, open-source editor based on Eclipse components. In addition to editing Dart programs, you can use Dart Editor to invoke the Dart-to-JavaScript compiler and launch Dart-based web apps.

Note: These instructions are also available for Windows and Mac. {% comment %} Linux. {% endcomment %}

    • Step 1: Download and install the software

    In this step, you'll install Dart Editor and, if necessary, a Java runtime environment.

    1. Download the Dart Editor ZIP file that matches your system architecture:
      • 32-bit Dart Editor
      • 64-bit Dart Editor

      If you don't know whether you have 32-bit or 64-bit Linux, use the command uname -a. If 64 is in the output, you have 64-bit Linux; otherwise, you have 32-bit Linux.

      Want the latest version? To try out the most recent, untested version of Dart Editor, download one of these instead:
      • 32-bit Dart Editor
      • 64-bit Dart Editor
    2. Unarchive the ZIP file. For example: 
       $ unzip DartBuild-linux.gtk.x86_64.zip

    3. Put the resulting directory somewhere convenient, such as /home/me/dart.

      We'll call this your Dart installation directory. This installation directory contains the DartEditor executable file and several subdirectories, including a samples directory.

    4. If you don't already have a Java runtime, download and install it. You can get JDK 7 from Java SE Development Kit 7 Downloads.

    Step 2: Launch the editor

    Go to your Dart installation directory, and double-click DartEditor.

    You should see the Dart Editor application window appear, looking something like this:
