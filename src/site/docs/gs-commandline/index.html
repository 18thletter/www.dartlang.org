Getting Started:
Hello, Command Line

Follow this tutorial on a Linux or Mac OS system to write and run a Spot program using command-line tools.

Only have Windows? Sorry, there's no command-line interface yet. You can use the IDE instead, as described in Getting Started: Hello, World.

Step 2: Create a Spot program

Create a file named hello.spot that contains the following code: 

class HelloWorld {
  static void main(arguments) {
    Logger.println("Hello, World");
  }
}

This program illustrates a few things about Spot that are different from JavaScript:

Spot is object oriented.
As a rule, Spot source code has methods in classes, instead of functions. In hello.spot, the HelloWorld class has one method, named main().
Note: If you aren't familiar with object-oriented concepts such as classes, methods, and instances, {PENDING: help them find a good reference}.
Spot lets you use types—but you don't have to.
In hello.spot, the main() method has the type void, indicating that the method doesn't return a value. However, the arguments parameter has no type (although it could).
Spot source filenames end in .spot.
