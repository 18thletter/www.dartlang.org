Isolates

All code in Dart runs in the context of an isolate. You can use additional isolates for concurrent programming, and to run third-party code more securely.

The isolates within an app don't share any state or memory. Instead, isolates communicate by passing messages. When an isolate sends a message, the receiving isolate gets a local copy of the original message's data.

Learn more about isolates in the library tour.
