interface Iterable<E>

Supertype:

  • Object

Subtypes:

  • Array<E>
  • Collection<E>
  • HashSet<E>
  • List<E>
  • Queue<E>
  • Set<E>
The Iterable<E> interface allows to get an Iterator<E> out of an Iterable<E> object. This interface is used by the for-in construct to iterate over an Iterable<E> object. The for-in construct takes an Iterable<E> object at the right-hand side, and calls its iterator method to get an Iterator<E> on it. A user-defined class that implements the Iterable<E> interface can be used as the right-hand side of a for-in construct.

Methods

Iterator<E> iterator()
Returns an Iterator<E> that iterates over this Iterable<E> object.
