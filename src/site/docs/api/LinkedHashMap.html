interface LinkedHashMap<K,V>

Supertype:

  • Object

Implemented interfaces:

  • HashMap<K,V>
Hash map version of the Map<K,V> interface that preserves insertion order.

Constructors

LinkedHashMap<K,V>()
Creates a map with the default implementation.
LinkedHashMap<K,V>.from(Map<K,V> other)
Creates a LinkedHashMap<K,V> that contains all key value pairs of other.
