Constructors

Duration([int days, int hours, int minutes, int seconds, int milliseconds])
The duration is the sum of all individual parts. This means that individual parts don't need to be less than the next-bigger unit. For example hours is allowed to have a value greater than 23. All individual parts are allowed to be negative. All arguments are by default 0.
