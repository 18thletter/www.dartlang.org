interface Match

Supertype:

  • Object
Match contains methods to manipulate a regular expression match. Iterables of Match objects are returned from RegExp matching methods. The following example finds all matches of a RegExp in a String and iterates through the returned iterable of Match objects.

    RegExp exp = const RegExp(@"(\w+)");
    String str = "Parse my string";
    Iterable<Match> matches = exp.allMatches(str, exp);
    for (Match m in matches) {
      String match = m.group(0);
      print(match);
    };
 
 The output of the example is:

    Parse
    my
    string

Methods

int end()
Returns the index in the string after the last character of the match.
String group(int group)
Returns the string matched by the given group. If group is 0, returns the match of the regular expression.
int groupCount()
Returns the number of groups in the regular expression.
List<E> groups(List<E> groups)
Returns the strings matched by groups. The order in the returned string follows the order in groups.
int start()
Returns the index in the string where the match starts.
