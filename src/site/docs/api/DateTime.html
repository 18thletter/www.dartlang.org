interface DateTime

Supertype:

  • Object

Implemented interfaces:

  • Comparable
DateTime is the public interface to a point in time.

Fields

Date date
Returns the date.
int day
Returns the date in the month [1..31].
int hours
Returns the number of hours [0..23].
int milliseconds
Returns the number of milliseconds [0...999].
int minutes
Returns the number of minutes [0...59].
int month
Returns the month in the year [1..12].
int seconds
Returns the number of seconds [0...59].
Time time
Returns the time.
TimeZone timeZone
int value
int weekday
Returns the week day [Date.MON..Date.SUN]
int year
Returns the year.

Constructors

DateTime(int year, int month, int day, int hours, int minutes, int seconds, int milliseconds)
Constructs a DateTime instance based on the individual parts, in the local time-zone.
DateTime.withTimeZone(int year, int month, int day, int hours, int minutes, int seconds, int milliseconds, TimeZone timeZone)
Constructs a DateTime instance based on the individual parts. timeZone may not be null .
DateTime.fromDateAndTime(Date date, Time time, TimeZone timeZone)
Constructs a DateTime instance based on the individual parts. If timeZone is null , then the local zone is used.
DateTime.now()
Constructs a new DateTime instance with current date time value. The timeZone of this instance is set to the local time-zone.
DateTime.fromString(String formattedString)
Constructs a new DateTime instance based on formattedString.
DateTime.fromEpoch(int value, TimeZone timeZone)
Constructs a new DateTime instance with the given time zone. The given timeZone must not be null . This constructor is the only one that doesn't need to be computations and which can therefore be const . The constructed DateTime represents 1970-01-01T00:00:00Z + valuems in the given timeZone.

Methods

DateTime add(Time other)
Returns a new DateTime with the time other added to this instance.
DateTime changeTimeZone(TimeZone targetTimeZone)
Returns a new DateTime in the given targetTimeZone time zone. The value of the new instance is equal to this.value . This call is equivalent to new DateTime.fromEpoch(this.value, targetTimeZone) .
Time difference(DateTime other)
Returns a Time with the difference of this and other.
bool isLocalTime()
Returns true if this DateTime is set to local time.
bool isUtc()
Returns true if this DateTime is set to UTC time. This is equivalent to this.timeZone.duration == 0 .
DateTime subtract(Time other)
Returns a new DateTime with the time other subtracted from this instance.
String toString()
Returns a human readable string for this instance. The returned string is constructed for the timeZone of this instance.
