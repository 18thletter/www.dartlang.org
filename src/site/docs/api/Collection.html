interface Collection<E>

Supertype:

  • Object

Implemented interfaces:

  • Iterable<E>

Subtypes:

  • Array<E>
  • HashSet<E>
  • List<E>
  • Queue<E>
  • Set<E>
The Collection<E> interface is the public interface of all collections.

Fields

int length
Returns the number of elements in this collection.

Methods

bool every(bool f(E))
Returns true if every elements of this collection satisify the predicate f. Returns false otherwise.
Collection<E> filter(bool f(E))
Returns a new collection with the elements of this collection that satisfy the predicate f. An element satisfies the predicate f if f(element) returns true.
void forEach(void f(E))
Applies the function f to each element of this collection.
bool isEmpty()
Returns true if there is no element in this collection.
bool some(bool f(E))
Returns true if one element of this collection satisfies the predicate f. Returns false otherwise.
