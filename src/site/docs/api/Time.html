Constructors

Time(int days, int hours, int minutes, int seconds, int milliseconds)
The time is the sum of all individual parts. This means that individual parts don't need to be less than the next-bigger unit. For example hours is allowed to have a value greater than 23. All individual parts are allowed to be negative.
Time.duration(int duration)
Constructs a time out of a duration. duration is in milliseconds.
