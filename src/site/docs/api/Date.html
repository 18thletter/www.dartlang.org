Constructors

Date(int year, int month, int day, int hours, int minutes, int seconds, int milliseconds)
Constructs a Date instance based on the individual parts, in the local time-zone.
Date.withTimeZone(int year, int month, int day, int hours, int minutes, int seconds, int milliseconds, TimeZone timeZone)
Constructs a Date instance based on the individual parts. timeZone may not be null .
Date.now()
Constructs a new Date instance with current date time value. The timeZone of this instance is set to the local time-zone.
Date.fromString(String formattedString)
Constructs a new Date instance based on formattedString.
Date.fromEpoch(int value, TimeZone timeZone)
Constructs a new Date instance with the given time zone. The given timeZone must not be null . This constructor is the only one that doesn't need to be computations and which can therefore be const . The constructed Date represents 1970-01-01T00:00:00Z + valuems in the given timeZone.

Methods

Date add(Duration duration)
Returns a new Date with the duration added to this instance.
Date changeTimeZone(TimeZone targetTimeZone)
Returns a new Date in the given targetTimeZone time zone. The value of the new instance is equal to this.value . This call is equivalent to new Date.fromEpoch(this.value, targetTimeZone) .
Duration difference(Date other)
Returns a Duration with the difference of this and other.
bool isLocalTime()
Returns true if this Date is set to local time.
bool isUtc()
Returns true if this Date is set to UTC time. This is equivalent to this.timeZone.isUtc() .
Date subtract(Duration duration)
Returns a new Date with the duration subtracted from this instance.
String toString()
Returns a human readable string for this instance. The returned string is constructed for the timeZone of this instance.
