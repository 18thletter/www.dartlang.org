interface HashMap<K,V>

Supertype:

  • Object

Implemented interfaces:

  • Map<K,V>

Subtypes:

  • LinkedHashMap<K,V>
Hash map version of the Map<K,V> interface. A HashMap<K,V> does not provide any guarantees on the order of keys and values in getKeys and getValues.

Constructors

HashMap<K,V>()
Creates a map with the default implementation.
HashMap<K,V>.from(Map<K,V> other)
Creates a HashMap<K,V> that contains all key value pairs of other.
