Dart numbers come in two flavors:

int
Integers of arbitrary size
double
Decimal 64-bit doubles, as specified by the IEEE 754 standard

Both int and double are subinterfaces of num. Using num

Methods such as ceil() and abs() are in the num interface, and not in Math.

{% pretty_code dart 0 %} // abs() gives the absolute value. print((-4).abs() == 4); //true // ceil() and floor() round up or down. print(3.25.ceil() == 4.0); //true print(3.25.floor() == 3.0); //true // round() rounds up for .5 or greater, otherwise it rounds down. print(3.25.round() == 3); //true print(3.89.round() == 4); //true print(3.5.round() == 4); //true print(3.49.round() == 3); //true // truncate() cuts off the decimal digits. print(3.24532.truncate() == 3.0); //true // toInt() converts a double to an int. print(3.24.toInt() == 3); //true {% endpretty_code %}

The num interface specifies the traditional bit shift, AND, and OR operators. For example:

{% pretty_code dart 0 %} print(3<<1 == 6); // 0011 << 1 == 0110 print(3>>1 == 1); // 0011 >> 1 == 0001 print((3|4) == 7); // 0011 | 0100 == 0111 {% endpretty_code %}

Summary of numbers

Dart supplies a num interface with two subinterfaces: int and double. Integers are arbitrary precision, meaning they have arbitrary size. Doubles are 64-bit decimals that follow IEEE 754 spec. The Math class has constants such as Math.PI and utilities such as parseDouble().
