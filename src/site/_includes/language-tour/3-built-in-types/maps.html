In general, a map is an object that associates keys to values. Dart support for maps is provided by map literals and the Map interface.

Here's a simple Dart map:

{% pretty_code dart 0 %} var gifts = { // a map literal // keys values "first" : "partridge", "second" : "turtledoves", "fifth" : "golden rings"}; {% endpretty_code %}

In map literals, each key must be a string. If you use a Map constructor, then you have more options: the key can be a string, a number, or any other object that implements the Hashable interface.

{% pretty_code dart 0 %} var map = new Map(); // use a Map constructor map[1] = "partridge"; // key is 1; value is "partridge" map[2] = "turtledoves"; // key is 2; value is "turtledoves" map[5] = "golden rings"; // key is 5; value is "golden rings" {% endpretty_code %}

A map value can be any object or null.

You add a new key-value pair to an existing map just as you would in JavaScript:

{% pretty_code dart 0 %} var gifts = { "first": "partridge" }; gifts["fourth"] = "calling birds"; // add a key-value pair {% endpretty_code %}

You retrieve a value from a map the same way you would in JavaScript:

{% pretty_code dart 0 %} var gifts = { "first": "partridge" }; print(gifts['first']); // partridge {% endpretty_code %}

If you look for a key that isn't in a map, you get a null in return. However, because values can be null, you might need to use a method such as containsKey() or putIfAbsent() to make sure you interpret null correctly.

{% pretty_code dart 0 %} var gifts = { "first": "partridge" }; print(gifts['fifth']); // null {% endpretty_code %}

Use .length to get the number of key-value pairs in the map:

{% pretty_code dart 0 %} var gifts = { "first": "partridge" }; gifts["fourth"] = "calling birds"; print(gifts.length); // 2 {% endpretty_code %}

To remove a key-value pair from a map, use the remove() method:

{% pretty_code dart 0 %} var gifts = { "first": "partridge" }; gifts["fourth"] = "calling birds"; gifts.remove('first'); print(gifts.length); // 1 print(gifts['first']); // null {% endpretty_code %}

You can copy a map using the Map.from() constructor:

{% pretty_code dart 0 %} var gifts = { "first": "partridge" }; var regifts = new Map.from(gifts); print(regifts['first']); // partridge {% endpretty_code %}

Iterating

You have a few choices for iterating through the contents of a map. Using the forEach() method gives you access to both the key and the value.

{% pretty_code dart 0 %} var gifts = { "first" : "partridge", "second": "turtledoves", "fifth" : "golden rings"}; gifts.forEach((k,v) => print('$k : $v')); {% endpretty_code %}
Note: Don't depend on forEach() returning the key-value pairs in a particular order.

If you are interested in just the keys or just the values, use getKeys() or getValues(), respectively. Both methods return a Collection object.

{% pretty_code dart 0 %} var gifts = {"first": "partridge", "second": "turtledoves"}; var values = gifts.getValues(); values.forEach((v) => print(v)); // partridge, turtledoves {% endpretty_code %}
Note: Map itself does not extend the Collection interface.
