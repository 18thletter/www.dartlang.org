In general, a map is an object that associates keys to values. Dart support for maps is provided by map literals and the Map interface.

Here's a simple Dart map:

{% highlight dart %} var gifts = { // A map literal // Keys Values "first" : "partridge", "second" : "turtledoves", "fifth" : "golden rings"}; {% endhighlight %}

In map literals, each key must be a string. If you use a Map constructor, then you have more options: the key can be a string, a number, or any other object that implements the Hashable interface.

{% highlight dart %} var map = new Map(); // Use a Map constructor. map[1] = "partridge"; // Key is 1; value is "partridge". map[2] = "turtledoves"; // Key is 2; value is "turtledoves". map[5] = "golden rings"; // Key is 5; value is "golden rings". {% endhighlight %}

A map value can be any object or null.

You add a new key-value pair to an existing map just as you would in JavaScript:

{% highlight dart %} var gifts = { "first": "partridge" }; gifts["fourth"] = "calling birds"; // Add a key-value pair. {% endhighlight %}

You retrieve a value from a map the same way you would in JavaScript:

{% highlight dart %} var gifts = { "first": "partridge" }; assert(gifts['first'] == 'partridge'); {% endhighlight %}

If you look for a key that isn't in a map, you get a null in return. However, because values can be null, you might need to use a method such as containsKey() or putIfAbsent() to make sure you interpret null correctly.

{% highlight dart %} var gifts = { "first": "partridge" }; assert(gifts['fifth'] == null); {% endhighlight %}

Use .length to get the number of key-value pairs in the map:

{% highlight dart %} var gifts = { "first": "partridge" }; gifts["fourth"] = "calling birds"; assert(gifts.length == 2); {% endhighlight %}

To remove a key-value pair from a map, use the remove() method:

{% highlight dart %} var gifts = { "first": "partridge" }; gifts["fourth"] = "calling birds"; gifts.remove('first'); assert(gifts.length == 1); assert(gifts['first'] == null); {% endhighlight %}

You can copy a map using the Map.from() constructor:

{% highlight dart %} var gifts = { "first": "partridge" }; var regifts = new Map.from(gifts); assert(regifts['first'] == 'partridge'); {% endhighlight %}

Iterating

You have a few choices for iterating through the contents of a map. Using the forEach() method gives you access to both the key and the value.

{% highlight dart %} var gifts = { "first" : "partridge", "second": "turtledoves", "fifth" : "golden rings"}; gifts.forEach((k,v) => print('$k : $v')); {% endhighlight %}
Tip: Don't depend on forEach() returning the key-value pairs in a particular order.

If you are interested in just the keys or just the values, use getKeys() or getValues(), respectively. Both methods return a Collection object.

{% highlight dart %} var gifts = {"first": "partridge", "second": "turtledoves"}; var values = gifts.getValues(); //Print partridge and turtledoves, but not necessarily in that order. values.forEach((v) => print(v)); {% endhighlight %}
Tip: Map itself does not extend the Collection interface.
