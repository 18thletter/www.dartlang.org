Dart Editor binaries are now available for Windows, Mac, and Linux. Dart Editor is a lightweight, open-source editor based on Eclipse components. In addition to editing Dart programs, you can use Dart Editor to invoke the Dart-to-JavaScript compiler and launch Dart-based web apps. Get started with the Dart Editor tutorial, where you can download the continuous or integration builds.
