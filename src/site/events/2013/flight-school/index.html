Dart helps you build modern apps that run across the modern web

Dart 1.0 provides a stable language, stable core libraries and tools such as a package manager, editor and dart2js compiler to help you be a productive web developer.

Dart has hit 1.0 and is now Flight Ready! Developer communities around the world are hosting Dart Flight School events during the month of February ‘14 to get people started on the language.

Organizer Resources

We’ll provide the materials you need to get your community onboarded with Dart. Here are some of the formats we will support for your Dart Flight School event:

  • Tech Talk

    Host a talk to learn more about what makes Dart a First-Class language.

    Download Tech Talk deck Tech Talk video

  • Code Lab

    Go from runway to take-off through a guided hands-on Code Lab.

    Download Code Lab worksheet GitHub Repository

  • Hackathon

    Create your own flight plan and use Dart to bring your ideas to the web.

    Download Hackathon guide
