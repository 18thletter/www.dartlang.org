First, let's take a look at the spirodraw declarations.

#library('spirodraw');

#import('dart:dom');
#source("ColorPicker.dart");
#resource("spirodraw.css");

A Dart script belongs to a libary declared with a name given by the #library directive. Dart scripts include other scripts using the #source directive and include other libraries with the #import directive. The #resource directive is currently not required, but ... The arguments to the import, source, and resource directives are URLs.

The spirodraw sample defines a library named 'spirodraw', which includes Spirodraw.dart, ColorPicker.dart, and the built-in Dart DOM library.
