Contact Us

If you have feedback, please go ahead and create a new issue in our issue tracker, or start a discussion on our misc group.

Here's a full list of the @dartlang.org groups:

misc
Join this list for general Dart discussions.
reviews
All Dart code reviews are sent to this list.
commits
All Dart commits to the source code repository are sent to this list.
bugs
All Dart issue tracker changes are sent to this list.

To get Dart's source code or read about how to build, test, and contribute code to Dart, go to the Dart open-source project:

dart.googlecode.com
