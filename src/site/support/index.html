Community & Support

Check out our groups! They're all @dartlang.org.

misc
Join this list for general Dart discussions.
reviews
All Dart code reviews are sent to this list.
commits
All Dart commits to the source code repository are sent to this list.
bugs
All Dart issue tracker changes are sent to this list.

All of Dart's source code, its issue tracker, and so on are available at dart.googlecode.com. If you have feedback, please go ahead and create a new issue in our issue tracker, or start a discussion on misc.
