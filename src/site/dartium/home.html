Dartium is the nickname of a Chromium build that has an embedded Dart virtual machine (VM). With this build, you can run and debug Dart web apps without compiling them to JavaScript.

Important: Dartium is a technical preview, and it might have security and stability issues.
Do not use Dartium as your primary browser!
