Intro

Spot is a language, libraries, and VM for structured web programming.

  • Batteries Included: language, core libraries, VM, IDE, and App Framework
  • Greatest Hits: class oriented, closures, array and hash literals, and more
  • New Tricks: optional typing, optional named arguments, named constructors, and more
  • Same code for the client and the server
